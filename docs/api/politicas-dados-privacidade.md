# Politicas de Dados e Privacidade - API

## Principios
- Minimizacao de dados no payload.
- Segregacao entre dados academicos, eventos e artefatos de midia.
- Uso de service role apenas no backend confiavel.

## Boas praticas
- Nao logar segredos.
- Limitar retorno de dados sensiveis em endpoints.
- Aplicar controle de acesso por JWT do Supabase.
