# Modelagem de Dados - Mobile (Consumo)

## Leitura
- classe, topicos, conteudos, atividades
- views de resumo/ranking
- conteudo_personalizado

## Escrita
- progresso em topico/conteudo/atividade/questao
- eventos em `eventos_aluno`

## Regra de negocio
Tempo e progresso devem ser persistidos durante o estudo ativo, para refletir corretamente no rank e nos graficos.
