# Politicas de Dados e Privacidade - Mobile

## Diretrizes
- Coleta de telemetria depende de consentimento.
- Tokens e segredos nao podem ser logados.
- Dados de estudo devem ser enviados apenas para endpoints autorizados.

## Recomendacoes
- Exibir transparência sobre coleta.
- Permitir funcionamento basico mesmo sem telemetria.
