# Estrutura de Banco - Resumo Executivo (Mobile)

## Objetivo
Dar visao rapida das entidades usadas pelo app.

## Essencial
- Progresso: `topico_aluno`, `conteudo_aluno`, `atividade_aluno`.
- Eventos: `eventos_aluno`.
- Ranking: `vw_rank_posicoes_por_classe`.
- Personalizacao: `conteudo_personalizado`.

## Regra
Consumir rank por view consolidada; nao depender de tabela derivada no app.
